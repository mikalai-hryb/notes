What is dependency injection?

In software engineering, dependency injection is a programming technique in which an object or function receives other objects or functions that it requires, as opposed to creating them internally.

dependency injection leads to loosely coupled parts of a program

Simplex communication is a communication channel that sends information in one direction only.

A half-duplex (HDX) system provides communication in both directions, but only one direction at a time, not simultaneously in both directions.

A full-duplex (FDX) system allows communication in both directions, and, unlike half-duplex, allows this to happen simultaneously.

Proxy server is a server application that acts as an intermediary between a client requesting a resource and the server providing that resource.
