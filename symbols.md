# Symbols

`!` - exclamation mark, bang, shriek
`#` - comment symbol, hash
`#!` - shebang(character sequence), hash-bang
`|` - pipe
`+` - addition
`-` - subtraction
`*` - multiplication
`/` - division
`%` - modulo
`^` - power/exponentiation
`==` - equal
`!=` - not-equal
`>`  - greater-than
`<`  - less-than
`>=` - greater-or-equal
`<=` - less-or-equal
`and`    - intersection
`or`     - union
`unless` - complement

`:` - colon
`;` - semicolon
`"` - double-quote
`'` - single-quote
\`  - backtick
`1,2,3...` - sequential numbers
`?` - question, question mark
